Museum Ticket Booking API

Timed-entry tickets, 4 time slots, 4 ticket types, visitor capacity per slot

Download Agent Reference Open Dashboard

Overview

Museum ticket booking API managing timed-entry visits across 4 daily time slots. Each slot has a 50-visitor capacity. Supports 4 ticket types (adult, child, senior, student) with different pricing. Booking tickets decrements available spots; cancelling frees them.

API Endpoints

GET /api/time-slots
List all time slots with their capacity.
GET /api/ticket-types
List all ticket types with prices.
GET /api/availability
Check visitor availability for a date. Returns remaining spots per slot.
ParameterTypeRequired
datedate (YYYY-MM-DD)Yes
time_slot_idintegerNo (returns single slot if provided)
visitorsintegerNo
GET /api/pricing
Get a price quote: price_per_person × num_visitors.
ParameterTypeRequired
ticket_typestringNo (default: adult)
visitorsintegerNo (default: 1)
POST /api/tickets
Book museum tickets for a specific date and time slot.
Body FieldTypeRequired
time_slot_idinteger (1–4)Yes
visit_datedate (YYYY-MM-DD)Yes
visitor_namestringYes
visitor_emailstringYes
num_visitorsinteger (≥1)Yes
ticket_typestringNo (default: adult)
GET /api/tickets
List all tickets. Filter by date and/or status.
GET /api/tickets/{id}
Get details of a specific ticket.
DELETE /api/tickets/{id}
Cancel a ticket. Frees up the spots.
