SkyMock Air — Flight Reservation API

Flight bookings across 6 US cities, 3 seat classes, seat availability tracking

Download Agent Reference Open Dashboard

Overview

Single-airline flight reservation API for the fictional SkyMock Air. Manages 15 flights between 6 US cities with 3 seat classes (economy, business, first) and dynamic seat availability tracking. Booking a flight decrements available seats; cancelling restores them.

API Endpoints

GET /api/destinations
List all served cities.
GET /api/flights
List all flights. Filter by origin, destination, date.
GET /api/flights/search
Search for available flights with enough seats.
ParameterTypeRequired
originstringYes
destinationstringYes
datedate (YYYY-MM-DD)No
passengersintegerNo
GET /api/flights/{id}
Get full details of a specific flight.
GET /api/pricing
Get a price quote by seat class and passenger count.
ParameterTypeRequired
flight_idintegerYes
seat_classstringNo (default: economy)
passengersintegerNo (default: 1)
POST /api/bookings
Create a new booking. Decrements available seats.
Body FieldTypeRequired
flight_idintegerYes
passenger_namestringYes
passenger_emailstringYes
num_passengersinteger (≥1)Yes
seat_classstringNo (default: economy)
GET /api/bookings
List all bookings. Filter by status.
GET /api/bookings/{id}
Get details of a specific booking.
DELETE /api/bookings/{id}
Cancel a booking. Restores available seats.
