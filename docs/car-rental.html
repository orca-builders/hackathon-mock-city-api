Car Rental API

16-vehicle fleet across 7 categories, date-range availability, daily rate pricing

Download Agent Reference Open Dashboard

Overview

Car rental API managing a fleet of 16 vehicles across 7 categories from economy to luxury. Supports date-range availability checking, daily rate pricing, and rental reservation management.

API Endpoints

GET /api/categories
List all vehicle categories with rate ranges.
GET /api/vehicles
List all vehicles. Filter by category and seats.
GET /api/vehicles/available
Find vehicles available for a date range.
ParameterTypeRequired
pickup_datedate (YYYY-MM-DD)Yes
return_datedate (YYYY-MM-DD)Yes
categorystringNo
seatsintegerNo
GET /api/vehicles/{id}
Get details of a specific vehicle.
GET /api/pricing
Get a price quote: daily_rate × num_days.
ParameterTypeRequired
vehicle_idintegerYes
pickup_datedate (YYYY-MM-DD)Yes
return_datedate (YYYY-MM-DD)Yes
POST /api/rentals
Create a new rental reservation.
Body FieldTypeRequired
vehicle_idintegerYes
customer_namestringYes
customer_emailstringYes
pickup_datedate (YYYY-MM-DD)Yes
return_datedate (YYYY-MM-DD)Yes
GET /api/rentals
List all rentals. Filter by status (confirmed / cancelled / completed).
GET /api/rentals/{id}
Get details of a specific rental.
DELETE /api/rentals/{id}
Cancel a rental.
