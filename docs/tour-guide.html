Tour Guide Booking API

12 guided tours across 6 categories with difficulty levels, group size limits, and per-person pricing

Download Agent Reference Open Dashboard

Overview

Tour guide booking API managing 12 guided tours across 6 categories (cultural, adventure, food, nature, nightlife, historical). Each tour has a maximum group size per day, difficulty level, duration, and per-person pricing. Bookings are date-specific with group capacity tracking.

API Endpoints

GET /api/categories
List tour categories with tour counts.
GET /api/tours
List all tours. Filter by category, difficulty, max_price, location.
GET /api/tours/{id}
Get full details of a specific tour.
GET /api/tours/available
Check availability for a tour on a specific date. Returns remaining spots.
ParameterTypeRequired
tour_idintegerYes
datedate (YYYY-MM-DD)Yes
guestsintegerNo
GET /api/pricing
Get a price quote: price_per_person × num_guests.
ParameterTypeRequired
tour_idintegerYes
guestsintegerNo (default: 1)
POST /api/bookings
Create a new booking for a tour on a specific date.
Body FieldTypeRequired
tour_idintegerYes
tour_datedate (YYYY-MM-DD)Yes
guest_namestringYes
guest_emailstringYes
num_guestsinteger (≥1)Yes
GET /api/bookings
List all bookings. Filter by status and/or date.
GET /api/bookings/{id}
Get details of a specific booking.
DELETE /api/bookings/{id}
Cancel a booking.
