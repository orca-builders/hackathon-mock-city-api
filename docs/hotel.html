Hotel Reservation API

Room reservations, date-range pricing with weekend surcharges, capacity management

Download Agent Reference Open Dashboard

Overview

Simple hotel reservation API designed for LLM / AI-agent consumption, with a dashboard for humans to monitor bookings. Manages 10 rooms across 3 types (single, double, suite) with date-range availability checking and automatic weekend surcharge pricing.

API Endpoints

GET /api/rooms
List all hotel rooms with type, capacity, and base price per night.
GET /api/rooms/available
Find rooms available for a given date range and party size.
ParameterTypeRequired
check_indate (YYYY-MM-DD)Yes
check_outdate (YYYY-MM-DD)Yes
guestsintegerNo
GET /api/pricing
Get a detailed price quote with per-night breakdown and weekend surcharges (Fri/Sat = 1.2x).
ParameterTypeRequired
room_idintegerYes
check_indate (YYYY-MM-DD)Yes
check_outdate (YYYY-MM-DD)Yes
POST /api/reservations
Create a new reservation.
Body FieldTypeRequired
room_idintegerYes
guest_namestringYes
guest_emailstringYes
check_indate (YYYY-MM-DD)Yes
check_outdate (YYYY-MM-DD)Yes
num_guestsinteger (≥1)Yes
GET /api/reservations
List all reservations. Optionally filter by status (confirmed / cancelled).
GET /api/reservations/{id}
Get details of a specific reservation.
DELETE /api/reservations/{id}
Cancel a reservation. Sets status to 'cancelled'.
