API Key Authentication

Every API endpoint is protected by an API key. Provide it in one of two ways:

# Header (recommended)
curl -H "X-API-Key: hotel-1-key-abc123" https://hacketon-18march-api.orcaplatform.ai/hotel-1/api/rooms

# Query parameter
curl "https://hacketon-18march-api.orcaplatform.ai/hotel-1/api/rooms?api_key=hotel-1-key-abc123"

For LLM / AI Agents

Each service exposes a self-describing schema endpoint at GET /<service>/api/schema that returns a JSON manifest with all endpoints, parameters, payloads, and domain values.

For comprehensive offline documentation, download the Agent Reference file from each service page. These agent.md files contain everything an AI agent needs: authentication, workflows, request/response schemas, seed data, and error handling.
